What's a "Discord bot"?

Discord bot is an application that we code inside Discord. That application can interact with the Discord features if we code it right. Discord bot can do banning, kicking, timing out, warning, etc! Discord bot can also be a game, for example economy bots (the bot type i usually make).


How to make one?

To make it, head on to this site. And then, see top-left. There's a button that says "New application", click it. After that, you'll see a form that asks you name and checking something. The name input means the name of the bot, I will make the name "ChillinBot!" here.

After doing those, you'll see this. Click on bot. There will be a button that says "Reset token". Don't worry, because that's normal. Now, Discord automatically creates a bot account once an application was made. Once you click "Reset token", you'll see this. Make sure to not share your token to other people, because if they get hands to them, they can inject any code to that bot anytime and anywhere. If you feel that your token is compromised, reset the token by clicking "Reset token" again and changing them on your code.

Ok I've done this step, then how to code my bot with node.js?

To code your bot, make a node.js server. There's a lot of options on where, but if you want to make a testing bot, choose using Replit. It's a free hosting place, and it also gives you FREE domain!

After logging in, create a new REPL and choose Node.js template..


After doing that, we're going to paste our token in a secret, a default enviromental variables in Replit.



After that, paste this setup code

      
const { Client, Events, GatewayIntentBits } = require('discord.js');
const token = process.env.token

const client = new Client({
  intents: [
    GatewayIntentBits.Guilds,
    GatewayIntentBits.GuildMessages,
    GatewayIntentBits.GuildMembers,
    GatewayIntentBits.MessageContent
  ]
});

client.once(Events.ClientReady, (c) => {
  console.log(`Ready! Logged in as ${c.user.tag}.`)
});

client.login(token)

At this point, if you run the server (by clicking button run), you should bring the bot online. To see, go back to Discord developer portal and invite your bot. To invite your bot, head to Oauth2 and click URL generator. There's a scope checklists. Check "bot" and "applications.commands". Give your bots permissions. Copy and paste the invite link to a browser and invite your bot into the server you want.
