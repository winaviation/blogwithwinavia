Make a Windows Bootleg | winaviation2


First, what is a Windows Bootleg?


Windows Bootleg is a unofficial release of Microsoft and is a modified Windows version. I even have my own one.


How to make your own Windows Bootleg


Winaviation2 will teach you how to make one!


  • First, you need a Windows Installation ISO which can be downloaded from official Microsoft's website.

    • Here I will use Windows 11 ISO


  • After you downloaded the ISO, extract it to a folder (name it anything but dont include space in it!)

  • You access the folder and go to sources, you will find install.esd or install.wim here. Open CMD as Administrator in sources.

  • Then you have to choose which Windows (11) version you will modify by choosing install.wim(esd) index using the command: Dism /Get-ImageInfo /imagefile:"insert install.wim(esd) file path here"

    • You will find some Windows version which was: Windows 11 Home, Windows 11 Pro, Windows 11 Pro N, Windows 11 Education... Check the index number of your Windows version and use the command below.


  • dism /export-image /SourceImageFile:"YourInstall.wim(esd)FilePath" /SourceIndex:YourIndexNumber /DestinationImageFile:YourCreatedFolderPath\install.wim(ThereWereNoEsd) /Compress:max /CheckIntegrity

  • Finally, you mount the new install.wim, create a folder called PATH in the folder that you created.

    • Use this command after you done the last step:


    DISM /Mount-image /imagefile:YourCreatedFolderPath\install.wim /Index:1 /MountDir:"YourCreatedFolderPath\PATH" /optimize


  • After done mounting then you are freely to edit the Windows files! Easy, right? NO!

  • Even when you mounted the wim by yourself then you still didnt got enough permission to edit it, TrustedInstaller keeps it all away. Search for youtube tutorials for bypass.

  • After you done modifying the install.wim, we will unmount.

    • Use the command: dism /unmount-Wim /MountDir:YourCreatedFolderPath\PATH /commit


  • Now you must delete or move the original install.wim(esd) to another location and then move the modified install.wim to sources.

  • Last step!

    • Now finally, we will build the new ISO. Winaviation2 recommended you to use MODWIN, a third party tool which uses to modify ISOs. Download version 3.2 here (also is for 64 bits computer only)


  • After you done downloading and opened the exe, go to C:\MODWIN and open modwin.bat

  • Then you press 7 and enter.

  • Copy and paste the ISO files which you extracted earlier to C:\MODWIN\ISO

  • Now in the modwin.bat window, you press 1 and then enter.

  • Then after it finished, finally, we're done! You will find yourself the new ISO called winmod.iso in C:\MODWIN\MOD.
