How to change Windows version to anything you want | winaviation2


Note: The current Windows you're using won't change even you change the Windows version. This means changing the Windows version doesn't affect the OS.


The software that is required:


  • Resource Hacker


    • How to change the Windows version.


    The result will shown in msinfo32 or winver.


  • First, we will open Resource Hacker, a 3rd-party software.
  • Click the open icon and navigate to C:\Windows\Branding\Basebrd\en-US\basebrd.dll.mui. Open that file.
  • Expand "String Table" and find the resource that contains the name of your current Windows version, for me, is Windows 10 Enterprise LTSC.
  • Now you can change the Windows version simply by just edit the the text.
  • Save the file with the name "basebrd.dll.mui" (overwrites the old file) after you finished changing.

    • Now when you open msinfo32 or winver, it will display your custom Windows version.



    Changing winver Windows logo


  • First, use any image editor and create your own custom logo (remember to save as .bmp and the image resolution needs to be 458 x 72)
  • Open Resource Hacker again. This time we will open C:\Windows\Branding\Basebrd\basebrd.dll
  • Select "Bitmap" and right click it, then click "Replace Bitmap".
  • Open the image that you created for the custom logo.
  • Now you will click "Replace".

    • Finally, you're done!

    Changing msconfig OS name


    When you open msconfig, you will see your default OS name, yes there is a way to change it. Normally you don't see any such options to rename, you must use cmd.exe


    Tutorial!


  • Open cmd.exe
  • Enter this command: bcdedit /set {current} description "x"

    • Replace x with the name of your custom Windows version. After that you are done!
