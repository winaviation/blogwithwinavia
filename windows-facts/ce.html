1. What is Windows CE?

Windows CE or Windows Embedded Compact is Microsoft's operating system for embedded devices and minimal computers. Windows CE is a different operating system and kernel. It is not a trimmed-down version of desktop Windows. It is not Windows XP Embedded which is NT-based. (According to Wikipedia)
